
# Todo Demo

参考自官方文档,一个运用 React 配合 Redux 开发的 Todo Demo

## 包括

- [x] [Webpack](https://webpack.github.io)
- [x] [React](https://facebook.github.io/react/)
- [x] [Redux](https://github.com/reactjs/redux)
- [x] [Babel](https://babeljs.io/)
- [x] TodoMVC example

## 运行

```
$ npm install
$ npm start
```

## 构建

```
$ npm run build
```
